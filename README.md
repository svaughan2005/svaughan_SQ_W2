# Project Title

svaughan_SQ_W2

## Setup and Interaction Instructions

The file can be run through its Github Pages link.

**Controls:**

- Move left/right: Arrow Keys or A/D
- Jump: Up Arrow or W
- Fall off the screen to reset to the start
- One of the platforms changes the character's hair colour! Find and land on it to activate this.
